# CS101 - Problem Solving and Python Programming (Draft)

Institution: Anurag University
Program: B.Tech CSE
Semester: 1
Credits: 4 (3-1-0)
Class Size: 72 students

## Course Outcomes (Draft)
1. Write Python programs using sequence, selection, and iteration.
2. Decompose problems into functions and reusable modules.
3. Implement core data structures (list, tuple, dict, set) for simple applications.
4. Apply basic testing and debugging techniques.

## Week-Wise Plan (Draft)
- Week 1: Computing basics, Python setup, variables, data types
- Week 2: Operators, input/output, conditional statements
- Week 3: Loops and pattern exercises
- Week 4: Functions and parameter passing
- Week 5: Strings and string methods
- Week 6: Lists and list comprehensions
- Week 7: Tuples, sets, dictionaries
- Week 8: File handling and exceptions
- Week 9: Mid-term practical prep
- Week 10: Simple OOP concepts
- Week 11: Mini project planning
- Week 12: Mini project implementation
- Week 13: Testing and debugging clinic
- Week 14: Viva and project demo

## Assessment Plan (Current)
- Internal 1 (theory): 15%
- Internal 2 (theory): 15%
- Assignment + lab record: 20%
- Lab practical: 20%
- End semester exam: 30%
